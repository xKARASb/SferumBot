Что бы пересылать все собщения из Сферум в телеграм!

#Что сейчас обрабатывает бот:
- Сообщения с медиа (фото, видео)
- Текстовые сообщения
- Пересланные сообщения (пока без медиа, т.к у меня где-то устекают ссылки в рекурсии)

# Доступные команды:
- /start начало и общий функционал
- /sferum инструкиция для Сферума

Что бы развернуть
